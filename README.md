# # 📊 Data Visualization using Seaborn & Bokeh

This project demonstrates the implementation of various data visualization techniques using **Seaborn** and **Bokeh** in Python.

---

## 🔹 Features

- Relational plots (scatter, relplot)
- Line plots for trend analysis
- Bar plots for comparison
- Box plots for distribution analysis
- Point plots
- Colour palette customization
- Interactive visualizations using Bokeh
- Handling deprecation warnings (circle ➝ scatter)

---

## 🔹 Technologies Used

- Python
- Seaborn
- Bokeh
- Matplotlib
- Google Colab

---

## 🔹 Sample Visualizations

- Bar Plot
- Box Plot
- Line Plot
- Scatter Plot (Bokeh)
- Interactive Charts

## 🔹 Key Learnings

- Customizing visual appearance using palettes
- Building interactive plots using Bokeh
- Writing clean and updated code
